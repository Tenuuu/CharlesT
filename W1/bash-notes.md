# Bash Notes

## File Paths

- Absolute - The whole (specific) path - /c/Users/RichardHawkins
- Relative - The referential path (from the current working directory, 
where is the target?) - ./Users/RichardHawkins

- "~" - the Home Directory - *We Can Modify THIS!!!
- "./" - The current working directory
- "../" - The parent directory / one level up

## Navigation

- cd - Change Directory
- ls - LiSt (folder contents)
- pwd - Print the Working Directory

## File/Folder Management

- mkdir - MaKe a new DIRectory
- touch - make a new file (or update the "last modified" timestamp!)
- rm - ReMove/delete a file
	- -r to make it Recursive, and delete folders (and all their contents!)
- rmdir - ReMove a DIRectory
- mv - MoVe a file (or directory)
- cp - CoPy a file (or a directory?)


## Utility

- clear - clear the screen
- echo - direct content to "the stream"
- stream redirection - take the steam, and point it to a new location
	- ">" - overwrite file content 
	- ">>" - append file content
- nano/vim - in terminal text editor (super light weight!)
